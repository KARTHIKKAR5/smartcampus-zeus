# 🍽️ The Last Meal Mile — Campus Food Delivery System

A fully working, production-grade internal campus food delivery platform for SR University.

## Stack
- **Frontend**: React + Vite + TailwindCSS + Leaflet.js
- **Backend**: FastAPI (Python)
- **Database**: PostgreSQL via Supabase
- **Auth**: Supabase Auth (JWT)
- **Maps**: OpenStreetMap + Leaflet
- **Route Optimization**: NetworkX (Dijkstra's Algorithm)
- **QR Codes**: qrcode + html5-qrcode
- **Real-time**: FastAPI WebSockets + Socket.IO
- **Analytics**: Chart.js + Pandas
- **Payments**: E-Wallet (internal ledger)

## Roles
- Student: Browse menu, place orders, track delivery, chat with runner, pay via wallet
- Runner: Accept orders, navigate routes, scan QR, chat with student
- Admin: Dashboard with analytics, manage runners, campus graph
- Canteen Owner: Manage menu, process orders, batch engine
